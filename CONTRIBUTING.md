# Contributing

## Setup

1. Enable Corepack and activate `pnpm`.
2. Install dependencies with `pnpm install`.
3. Confirm `cargo test` and `pnpm test` pass before opening a PR.

## Branches

- `feat/<topic>`
- `fix/<topic>`
- `docs/<topic>`

## Standards

- Use Conventional Commits.
- Add or update at least one automated test for behavior changes.
- Keep PRs focused on a single change.
